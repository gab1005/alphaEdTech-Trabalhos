Formas de implementar CSS no HTML

INLINE (em linha):
dessa forma a estilização vai dentro da tag.
<div style="background: red; width: 700px; color:black"> </div>

INTERNAL (interno):
nesse caso o comando de estilização fica no < head >
existem duas formas de trabalhar o CSS internal, dentro da tag < style > colocada na tag < head > se escreve o nome das tags que deseja alterar ou ou nome da classe ou id

escrevendo o nomes das tags diretamente:

< style type="text/css">
h4 p {
color: rgba(50, 200, 50, 0.8);
}

h4 p b {
color: rgba(255, 255, 255);
}
</style>

por classe
nesse caso se passa o atributo class="nome-da-classe"
e no style dentro de head se coloca o estilo "apontando" para a classe.

na tag: <p class="texto01">.

no head:

.texto01 {
color: crimson;
background: chocolate;
}

.texto01 i {
color: white;
}

por id:
semelhante ao classe, o que muda é que pode existir apenas um id no codigo, via de regra, podem surgir conflitos se ele não for usado de forma correta, tanto que se recomenda o uso de classes na maioria da situações.

na tag <p id="texto-id-01">.

no style dentro de head:

#texto-id-01 {
color: rgba(50,50,200,0.7);
}

EXTERNAL (externo):
até agora, todos os CSS foram colocados no proprio arquivos HTML a ser estilizado, agora no external o CSS fica em outro arquivo com extenção ".css", o arquivo a ser estilizado html chama o ".css" e acontece a estilização da pagina, da mesma forma do internal, se pode alterar os objetos escrevendo diretamente as tags, ou por meio de "apontar" a classe ou id a ser modificada.

na tag < p class="texto02"> .

na arquivo ".ccs" externo:
.texto02 {
color:aqua;
background: black;
}

.texto02 b {
color: white;
}
